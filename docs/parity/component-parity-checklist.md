# Component Parity Checklist

- `PageTitle`: same heading/divider rhythm as app.
- `FormField`: label above input, compact error text below, same rounded input feel.
- `StationCard`: frosted, rounded, top highlight, voucher chip, status chip, directions CTA.
- `PlanCard`: icon block, price, duration cell, energy cell, bottom CTA.
- `SessionCard`: compact status pill + metric rows + buy more / stop actions.
- `AccountActionRow`: title + short description + chevron only.
- `ConfirmModal`: short row summary + primary/secondary CTA.
- `ToastOverlay`: centered success/error overlay, not a web toast bar.
